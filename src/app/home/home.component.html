Escape The Picture

Une aventure artistique

Bienvenue dans le monde de l'art

Vous venez d'être embauché(e) comme conservateur/conservatrice dans un prestigieux musée d'art moderne. Votre première mission : organiser une exposition exceptionnelle sur l'un des plus grands artistes du 20ème siècle.

Mais il y a un problème... Les archives du musée sont en désordre et les informations essentielles sur l'artiste ont été dispersées dans différentes salles. Sans ces informations, l'exposition ne pourra pas avoir lieu !

Votre mission est de traverser les salles du musée, résoudre les énigmes cachées dans les œuvres et rassembler tous les éléments nécessaires pour reconstituer l'histoire de cet artiste légendaire.

Êtes-vous prêt(e) à relever le défi ?

Instructions

  • Explorez chaque salle en cliquant sur les zones interactives
  • Collectez les indices pour résoudre les énigmes
  • Découvrez l'identité de l'artiste mystère
  • Utilisez les indices avec attention : certains peuvent être trompeurs...
